# MTRPZ LAB 1
Program on JavaScript which converts markdown syntax to html using command line interface.

## Usage

1. Clone a git repository:

2. Install dependencies:
  ```shell
  $ npm i
  ```

3. Run
```shell
$ node index.js --file path/to/md/file --out path/to/html/file
```
